哇，摆弄了两天，终于弄好了自己的博客，以后就要经常发一些读书笔记，项目过程了~

首先讲讲自己的这个博客建立过程。

本来我是想自己弄个服务器搭建的，但是国内的网站竟然要备案！！先前没有料想到，也算是自己没有
做好准备吧，花了好久的时间在那个云服务器里面配置IIS、php、mysql、wordpress等等等等的环境，
这些真的花了好多时间，windows server 2012真的不会用，好多东西都很难去在线安装，就那个微
软的web平台的安装软件下载的官方网站结果被IE增强的安全配置一直提示无法下载（都是自家人，这
是闹哪样啊。。。）然后本地到云服务器转移文件也超级慢，反正全部配置好几乎花了我全部周末的时间
（就因为沉迷这个，没有及时做好答辩的ppt），然后十分兴奋的准备用域名打开博客时，告诉我要备
案，整个人是崩溃的，一看流程超级复杂，就放弃了。。就这样转到了github。
昨晚今早我还在用hugo搭建，结果不太理想，操作也不是很熟练，界面也不是很好看，也放弃了，
就选择了这个hexo，因为有了之前的经验而且这次网上的资料非常充分，Git Bash的操作也算是比较
熟练了（而且运用hugo充分锻炼了我命令行的基础操作），这次非常快，前前后后，从换主题到插件
配置再到细节处理用了两三个小时吧，个人还是比较满足了，哈哈哈哈，啊噗啊噗！
